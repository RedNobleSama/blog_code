### 新建git仓库
> git init

### 上传修改文件
> git add

### 上传文件描述
> git commit -m "xxx"

### 上传分支
> git push origin master

### 拉去更新分支
> git pull origin master

### 更新分支
>git add
>git commit -m "xxx"
>git push origin master
